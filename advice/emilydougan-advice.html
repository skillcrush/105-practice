Advice for Newbies

Move fast, break things

Don't be afraid to dive right in, even when you feel like you're not ready. You're never going to be ready! Life is a long learning journey. Try that new language. Build that crappy project. Accidentally delete everything. Build it all again. Rinse and repeat.

All Practice is Not Created Equally

Don't practice with the TV on. Don't practice with 35 tabs open. And definitely don't add things to your amazon cart when you're trying to study. Learning takes deep focus, so if you're going to dive in, you might as well go deep. Don't waste time splashing around the shallow end of the pool or you'll never learn how to swim.
