My Advice to Programmers

Stick to it!

Make a commitment to be on the job daily. Even if you only put in five minutes a day, it's still progress, and keeps you in the habit! Many things you learn will be hard at first, but with practice become much easier. So keep up the good work!

Do the research!

If you're stuck or need help, check out sites like the Mozilla Developer Network for helpful tips and tricks. There are also a number of coding forums and blogs around the web. It's helpful to search these since other people might have already solved a similar problem.

Go the extra mile!

Do the best you can every time. Others will know you as someone who takes their idea to the next level. If you have the skills, use them!

Hot Tips:

  • 1. Content, then styles.

    It can be easy to get distracted by styling. But it's more efficient to style after you have the content.

  • 2. Comment the code!

    Your code may make sense to you right now, but how about a few years into the future? And what if others want to understand it?

  • 3. Practice, practice, practice.

    When you learn a something new, try it out! Dont't be afraid to mess up. It's also a good idea to make a cheatsheet for ideas and concepts. It will be handy in the future!

  • 4. Save it often.

    One of the most frustrating scenarios is to make a bunch of progress, but lose it because you didn't save the code. So save it often. (And don't forget about Git and GitHub!)

  • 5. Validate your code!

    If you have a feeling that something is incorrect, plug the code into a validator like the W3C HTML Validator. It will catch errors that you might not see.
